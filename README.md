# 番茄小说OCR下载器 - 浏览器插件

使用OCR技术自动下载番茄小说网的小说内容。

## 功能特点

- 自动滚动页面并截图
- 使用Tesseract.js进行OCR文字识别
- 自动翻页下载多章节
- 一键导出TXT文件
- 实时进度显示
- 纯前端实现，无需后端服务

## 安装方法

### 1. 下载插件文件

确保你有以下文件结构：
```
browser-extension/
├── manifest.json
├── background.js
├── content.js
├── popup.html
├── popup.js
├── lib/
│   └── tesseract.min.js
└── icons/
    ├── icon16.png
    ├── icon48.png
    └── icon128.png
```

### 2. 加载到Chrome浏览器

1. 打开Chrome浏览器，输入 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `browser-extension` 文件夹
5. 插件安装成功！

## 使用方法

### 基本使用

1. 打开番茄小说网的小说阅读页面
2. 点击浏览器工具栏上的插件图标
3. 设置要下载的章节数（默认10章）
4. 点击"开始下载"按钮
5. 等待自动截图和OCR识别完成
6. 完成后自动下载TXT文件

### 注意事项

- 首次使用需要下载OCR语言模型（约10MB），请耐心等待
- 下载过程中请勿关闭页面或切换标签
- 建议在网络稳定的环境下使用
- OCR识别需要一定时间，请耐心等待

## 文件说明

| 文件 | 说明 |
|------|------|
| `manifest.json` | 插件配置文件 |
| `background.js` | 后台服务脚本，处理截图和下载 |
| `content.js` | 内容脚本，在小说页面执行 |
| `popup.html/js` | 弹出窗口界面和逻辑 |
| `lib/tesseract.min.js` | Tesseract.js OCR库 |

## 常见问题

### Q: OCR识别准确率低？
A: Tesseract.js的中文识别准确率不如PaddleOCR，建议：
- 确保页面文字清晰可见
- 调整浏览器缩放比例为100%
- 使用更高分辨率的屏幕

### Q: 下载速度很慢？
A: 
- OCR处理需要时间，每章约30-60秒
- 可以减少每章的滚动次数
- 关闭其他占用资源的标签页

### Q: 无法识别某些章节？
A:
- 检查网页是否正常加载
- 刷新页面后重试
- 检查选择器配置是否正确
